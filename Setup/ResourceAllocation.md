# Resource Allocation

## Host System Constraints
CPU: 1.8 GHz

## Allocation Strategy
Run one VM at a time for stability

## Kali Linux
- RAM: 2048 MB
- CPU: 1 core

## Windows 10
- RAM: 4096 MB (when running alone)
- CPU: 1 core

## Purpose
Ensure smooth performance without overloading host system
